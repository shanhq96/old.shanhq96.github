1. 基本数据类型


	基本类型，或者叫内置类型，是java中不同于类的特殊类型。它们是我们编程中使用最频繁的类型。共有八种，它们分别有对应的包装类。
	基本类型又分三类，字符类型*char*，布尔类型*boolean*以及数值类型*byte*、*short*、*int*、*long*、*float*、*double*。其中数值类型又可分为整数类型*byte*、*short*、*int*、*long*和浮点数类型*float*、*double*。
